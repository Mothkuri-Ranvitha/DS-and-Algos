# Duplicates & Uniqueness

> Detect repeated or unique elements.

---

## Core Idea

Use HashSet to track seen values.

---

## Recognition Signals

* Duplicate exists
* Unique elements
* Distinct count
* Seen before

---

## Mental Model

```text
Seen Before?

YES → Duplicate

NO → Add to Set
```

---

## Generic Template

```java
HashSet<Integer> set =
        new HashSet<>();

for(int num : nums) {

    if(set.contains(num))
        return true;

    set.add(num);
}
```

## Example Problems

### LC217 - Contains Duplicate

```java
class Solution {

    public boolean containsDuplicate(
            int[] nums) {

        HashSet<Integer> set =
                new HashSet<>();

        for(int num : nums) {

            if(set.contains(num))
                return true;

            set.add(num);
        }

        return false;
    }
}
```

Time: O(N)

Space: O(N)

### LC128 - Longest Consecutive Sequence

Idea:

Start sequence only when previous number absent.

```java
class Solution {

    public int longestConsecutive(
            int[] nums) {

        HashSet<Integer> set =
                new HashSet<>();

        for(int num : nums)
            set.add(num);

        int longest = 0;

        for(int num : set) {

            if(!set.contains(num-1)) {

                int current = num;
                int streak = 1;

                while(set.contains(
                        current+1)) {

                    current++;
                    streak++;
                }

                longest =
                    Math.max(
                        longest,
                        streak
                    );
            }
        }

        return longest;
    }
}
```

Time: O(N)

Space: O(N)
