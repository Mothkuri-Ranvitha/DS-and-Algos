# 1. LC841 - Keys and Rooms

## Pattern

* Reachability
* Basic Graph DFS

## Recognition

* Can visit all nodes?
* Reach every room?
* Unlock using keys

## Core Intuition

Start from room `0` and use DFS to visit every reachable room. If all rooms are visited, return `true`.

## Algorithm

1. Create `visited[]`.
2. DFS from room `0`.
3. Visit all keys inside each room.
4. Check if every room is visited.

## Java

```java
class Solution {

    public boolean canVisitAllRooms(List<List<Integer>> rooms) {

        boolean[] visited = new boolean[rooms.size()];

        dfs(0, rooms, visited);

        for (boolean room : visited)
            if (!room)
                return false;

        return true;
    }

    private void dfs(int room,
                     List<List<Integer>> rooms,
                     boolean[] visited) {

        visited[room] = true;

        for (int key : rooms.get(room)) {

            if (!visited[key])
                dfs(key, rooms, visited);
        }
    }
}
```

## Complexity

* Time : **O(V + E)**
* Space : **O(V)**

## Common Mistakes

* Forgetting `visited[]`
* Starting DFS from every room

## One-Line Revision

> DFS from room `0`; if every room is visited, return `true`.

# 2. LC547 - Number of Provinces

## Pattern

* Connected Components

## Recognition

* Count groups
* Connected cities
* Number of components

## Core Intuition

Every DFS visits one complete province. Count how many DFS calls are required.

## Algorithm

1. Iterate every city.
2. If not visited:

   * DFS
   * Increment answer.
3. Return count.

## Java

```java
class Solution {

    public int findCircleNum(int[][] isConnected) {

        int n = isConnected.length;
        boolean[] visited = new boolean[n];
        int provinces = 0;

        for (int i = 0; i < n; i++) {

            if (!visited[i]) {

                dfs(i, isConnected, visited);
                provinces++;
            }
        }

        return provinces;
    }

    private void dfs(int city,
                     int[][] graph,
                     boolean[] visited) {

        visited[city] = true;

        for (int next = 0; next < graph.length; next++) {

            if (graph[city][next] == 1 &&
                !visited[next]) {

                dfs(next, graph, visited);
            }
        }
    }
}
```

## Complexity

* Time : **O(V²)**
* Space : **O(V)**

## Common Mistakes

* Forgetting disconnected components

## One-Line Revision

> Number of DFS calls = Number of Provinces.

# 3. LC797 - All Paths From Source to Target

## Pattern

* DFS + Backtracking

## Recognition

* Find every path
* Generate all paths
* DAG traversal

## Core Intuition

Maintain the current path. When destination is reached, save the path. Backtrack after returning.

## Algorithm

1. Add current node.
2. If destination:

   * Save path.
3. DFS all neighbors.
4. Remove current node (backtrack).

## Java

```java
class Solution {

    List<List<Integer>> ans = new ArrayList<>();

    public List<List<Integer>> allPathsSourceTarget(int[][] graph) {

        List<Integer> path = new ArrayList<>();

        dfs(0, graph, path);

        return ans;
    }

    private void dfs(int node,
                     int[][] graph,
                     List<Integer> path) {

        path.add(node);

        if (node == graph.length - 1) {

            ans.add(new ArrayList<>(path));
        } else {

            for (int next : graph[node]) {

                dfs(next, graph, path);
            }
        }

        path.remove(path.size() - 1);
    }
}
```

## Complexity

* Time : **O(2^N)** (depends on number of paths)
* Space : **O(N)**

## Common Mistakes

* Forgetting backtracking
* Adding the same path reference

## One-Line Revision

> Add → DFS → Remove (Backtracking).

# 4. LC200 - Number of Islands

## Pattern

* Grid DFS
* Flood Fill

## Recognition

* Islands
* Connected land
* Grid traversal

## Core Intuition

Whenever land (`1`) is found, run DFS to sink the entire island. Each DFS represents one island.

## Algorithm

1. Traverse grid.
2. If land:

   * DFS
   * Increment answer.
3. Mark visited land as water.

## Java

```java
class Solution {

    public int numIslands(char[][] grid) {

        int islands = 0;

        for (int r = 0; r < grid.length; r++) {

            for (int c = 0; c < grid[0].length; c++) {

                if (grid[r][c] == '1') {

                    dfs(grid, r, c);
                    islands++;
                }
            }
        }

        return islands;
    }

    private void dfs(char[][] grid,
                     int r,
                     int c) {

        if (r < 0 ||
            c < 0 ||
            r == grid.length ||
            c == grid[0].length ||
            grid[r][c] == '0')
            return;

        grid[r][c] = '0';

        dfs(grid, r - 1, c);
        dfs(grid, r + 1, c);
        dfs(grid, r, c - 1);
        dfs(grid, r, c + 1);
    }
}
```

## Complexity

* Time : **O(R × C)**
* Space : **O(R × C)**

## Common Mistakes

* Not marking visited
* Diagonal traversal (only 4 directions)

## One-Line Revision

> Every DFS sinks one complete island.

# 5. LC695 - Max Area of Island

## Pattern

* Grid DFS Returning Integer

## Recognition

* Largest island
* Maximum area
* Connected region size

## Core Intuition

Each DFS returns the area of its island. Keep the maximum.

## Algorithm

1. Traverse grid.
2. If land:

   * Area = DFS()
3. Answer = max(answer, area)

## Java

```java
class Solution {

    public int maxAreaOfIsland(int[][] grid) {

        int maxArea = 0;

        for (int r = 0; r < grid.length; r++) {

            for (int c = 0; c < grid[0].length; c++) {

                if (grid[r][c] == 1) {

                    maxArea = Math.max(maxArea,
                            dfs(grid, r, c));
                }
            }
        }

        return maxArea;
    }

    private int dfs(int[][] grid,
                    int r,
                    int c) {

        if (r < 0 ||
            c < 0 ||
            r == grid.length ||
            c == grid[0].length ||
            grid[r][c] == 0)
            return 0;

        grid[r][c] = 0;

        return 1
                + dfs(grid, r - 1, c)
                + dfs(grid, r + 1, c)
                + dfs(grid, r, c - 1)
                + dfs(grid, r, c + 1);
    }
}
```

## Complexity

* Time : **O(R × C)**
* Space : **O(R × C)**

## Common Mistakes

* Returning `0` incorrectly
* Forgetting the `+1` for the current cell

## One-Line Revision

> DFS returns island area; take the maximum.

# 6. LC130 - Surrounded Regions

## Pattern

* Boundary DFS

## Recognition

* Capture surrounded regions
* Boundary cells
* Safe vs Unsafe regions

## Core Intuition

Instead of finding surrounded `'O'`, start DFS from **boundary `'O'`** and mark them as safe. Flip the remaining `'O'` to `'X'`.

## Algorithm

1. DFS from all boundary `'O'`.
2. Mark safe cells as `'#'`.
3. Traverse grid:

   * `'O'` → `'X'`
   * `'#'` → `'O'`

## Java

```java
class Solution {

    public void solve(char[][] board) {

        int rows = board.length;
        int cols = board[0].length;

        for (int r = 0; r < rows; r++) {
            dfs(board, r, 0);
            dfs(board, r, cols - 1);
        }

        for (int c = 0; c < cols; c++) {
            dfs(board, 0, c);
            dfs(board, rows - 1, c);
        }

        for (int r = 0; r < rows; r++) {
            for (int c = 0; c < cols; c++) {

                if (board[r][c] == 'O')
                    board[r][c] = 'X';
                else if (board[r][c] == '#')
                    board[r][c] = 'O';
            }
        }
    }

    private void dfs(char[][] board, int r, int c) {

        if (r < 0 || c < 0 ||
            r == board.length ||
            c == board[0].length ||
            board[r][c] != 'O')
            return;

        board[r][c] = '#';

        dfs(board, r - 1, c);
        dfs(board, r + 1, c);
        dfs(board, r, c - 1);
        dfs(board, r, c + 1);
    }
}
```

## Complexity

* **Time:** O(R × C)
* **Space:** O(R × C)

## Common Mistakes

* Starting DFS from interior cells
* Forgetting to restore `'#'`

## One-Line Revision

> Mark boundary-connected `'O'` as safe, flip the rest.

# 7. LC1254 - Number of Closed Islands

## Pattern

* DFS Returning Boolean

## Recognition

* Closed island
* Surrounded by water
* Return true/false

## Core Intuition

DFS returns whether the current island is closed. If any path reaches outside the grid, the island is not closed.

## Algorithm

1. Traverse all land.
2. DFS returns `boolean`.
3. Count only if DFS returns `true`.

## Java

```java
class Solution {

    int count = 0;

    public int closedIsland(int[][] grid) {

        for (int r = 0; r < grid.length; r++) {

            for (int c = 0; c < grid[0].length; c++) {

                if (grid[r][c] == 0 && dfs(grid, r, c))
                    count++;
            }
        }

        return count;
    }

    private boolean dfs(int[][] grid, int r, int c) {

        if (r < 0 || c < 0 ||
            r == grid.length ||
            c == grid[0].length)
            return false;

        if (grid[r][c] == 1)
            return true;

        grid[r][c] = 1;

        boolean up = dfs(grid, r - 1, c);
        boolean down = dfs(grid, r + 1, c);
        boolean left = dfs(grid, r, c - 1);
        boolean right = dfs(grid, r, c + 1);

        return up && down && left && right;
    }
}
```

## Complexity

* **Time:** O(R × C)
* **Space:** O(R × C)

## Common Mistakes

* Using `||` instead of `&&`
* Returning `true` at the boundary

## One-Line Revision

> Every DFS returns whether the island is completely enclosed.

# 8. LC1020 - Number of Enclaves

## Pattern

* Boundary Elimination

## Recognition

* Escape boundary
* Enclaves
* Remove boundary land

## Core Intuition

Remove every land cell connected to the boundary. Whatever land remains is an enclave.

## Algorithm

1. DFS from boundary land.
2. Convert boundary-connected land to water.
3. Count remaining land.

## Java

```java
class Solution {

    public int numEnclaves(int[][] grid) {

        int rows = grid.length;
        int cols = grid[0].length;

        for (int r = 0; r < rows; r++) {
            dfs(grid, r, 0);
            dfs(grid, r, cols - 1);
        }

        for (int c = 0; c < cols; c++) {
            dfs(grid, 0, c);
            dfs(grid, rows - 1, c);
        }

        int count = 0;

        for (int[] row : grid)
            for (int cell : row)
                if (cell == 1)
                    count++;

        return count;
    }

    private void dfs(int[][] grid, int r, int c) {

        if (r < 0 || c < 0 ||
            r == grid.length ||
            c == grid[0].length ||
            grid[r][c] == 0)
            return;

        grid[r][c] = 0;

        dfs(grid, r - 1, c);
        dfs(grid, r + 1, c);
        dfs(grid, r, c - 1);
        dfs(grid, r, c + 1);
    }
}
```

## Complexity

* **Time:** O(R × C)
* **Space:** O(R × C)

## Common Mistakes

* Counting before removing boundary land

## One-Line Revision

> Remove boundary-connected land, count the remaining cells.

# 9. Detect Cycle in Undirected Graph

## Pattern

* Parent Tracking

## Recognition

* Detect cycle
* Undirected graph
* Valid tree

## Core Intuition

If DFS reaches a visited node that is **not the parent**, a cycle exists.

## Algorithm

1. DFS(node, parent)
2. Visit neighbors.
3. If visited neighbor ≠ parent → cycle.

## Java

```java
boolean dfs(int node,
            int parent,
            boolean[] visited,
            ArrayList<ArrayList<Integer>> adj){

    visited[node]=true;

    for(int nei:adj.get(node)){

        if(!visited[nei]){

            if(dfs(nei,node,visited,adj))
                return true;
        }

        else if(nei!=parent){

            return true;
        }
    }

    return false;
}
```

## Complexity

* **Time:** O(V + E)
* **Space:** O(V)

## Common Mistakes

* Forgetting parent parameter
* Starting DFS from only one node

## One-Line Revision

> Visited neighbor other than parent = cycle.

# 10. Detect Cycle in Directed Graph

## Pattern

* Recursion Stack

## Recognition

* Directed graph
* Circular dependency
* Course Schedule

## Core Intuition

A cycle exists if DFS reaches a node already present in the current recursion stack.

## Algorithm

1. Maintain `visited[]`.
2. Maintain `pathVisited[]`.
3. Back edge ⇒ cycle.

## Java

```java
boolean dfs(int node,
            ArrayList<ArrayList<Integer>> adj,
            boolean[] visited,
            boolean[] pathVisited){

    visited[node]=true;
    pathVisited[node]=true;

    for(int nei:adj.get(node)){

        if(!visited[nei]){

            if(dfs(nei,adj,visited,pathVisited))
                return true;
        }

        else if(pathVisited[nei]){

            return true;
        }
    }

    pathVisited[node]=false;

    return false;
}
```

## Complexity

* **Time:** O(V + E)
* **Space:** O(V)

## Common Mistakes

* Using parent tracking
* Forgetting `pathVisited[node] = false`

## One-Line Revision

> A back edge to a node in the current DFS path indicates a cycle.
