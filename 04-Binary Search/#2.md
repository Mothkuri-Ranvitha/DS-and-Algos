# Lower Bound

> Find the first position where a value can appear.

---

## Core Idea

Not searching for target.

Searching for:

```text
First Valid Position
```

Example:

```text
1 2 2 2 3

Find First 2

Answer = Index 1
```

---

## Recognition Signals

Look for:

* First Occurrence
* First Position
* Search Insert Position
* Left Boundary

---

## Mental Model

```text
Found Target?

Don't Stop.

Move Left.
```

---

## Template

```java
int low = 0;
int high = nums.length;

while(low < high){

    int mid =
        low + (high-low)/2;

    if(nums[mid] >= target)
        high = mid;
    else
        low = mid + 1;
}

return low;
```
