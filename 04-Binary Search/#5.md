# Binary Search On Answer

> Search the answer space instead of the array.

---

## Core Idea

We are not searching:

```text
Array Element
```

We are searching:

```text
Minimum Valid Answer

or

Maximum Valid Answer
```

---

## Recognition Signals

Look for:

* Minimum Possible
* Maximum Possible
* Smallest Valid
* Largest Valid
* Capacity
* Speed
* Distance

---

## Mental Model

```text
Can Answer = Mid Work?

YES
↓
Try Better Answer

NO
↓
Try Larger Answer
```

---

## Generic Template

```java
while(low < high){

    int mid =
        low + (high-low)/2;

    if(canDo(mid))
        high = mid;

    else
        low = mid + 1;
}
```
