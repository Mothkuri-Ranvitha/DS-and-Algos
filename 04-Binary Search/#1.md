# Classic Binary Search

> Find a target in a sorted array.

---

## Core Idea

Array is sorted.

Check middle element.

```text
Target < Mid → Go Left

Target > Mid → Go Right

Target = Mid → Found
```

Every step removes half the search space.

---

## Recognition Signals

Look for:

* Sorted Array
* Search Target
* Find Position
* Exact Match

---

## Mental Model

```text
Sorted Array
      ↓
Check Middle
      ↓
Discard Half
      ↓
Repeat
```

---

## Generic Template

```java
int low = 0;
int high = nums.length - 1;

while(low <= high){

    int mid =
        low + (high-low)/2;

    if(nums[mid] == target)
        return mid;

    if(nums[mid] < target)
        low = mid + 1;
    else
        high = mid - 1;
}

return -1;
```
