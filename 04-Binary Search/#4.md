# Rotated Sorted Array

> Array is sorted but rotated.

---

## Core Idea

Example:

```text
4 5 6 7 0 1 2
```

Observation:

```text
At least one side
is always sorted.
```

Use that sorted half to decide.

---

## Recognition Signals

Look for:

* Rotated Array
* Circular Sorted Array
* Pivot Rotation

---

## Mental Model

```text
Find Sorted Half
      ↓
Check Target
      ↓
Discard Other Half
```

---
