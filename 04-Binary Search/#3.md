# Upper Bound

> Find the last valid position.

---

## Core Idea

Find:

```text
Last Occurrence
```

Example:

```text
1 2 2 2 3

Last 2

Answer = Index 3
```

---

## Recognition Signals

Look for:

* Last Occurrence
* Right Boundary
* Last Position

---

## Mental Model

```text
Found Target?

Move Right.
```

---

## Template

```java
int low = 0;
int high = nums.length;

while(low < high){

    int mid =
        low + (high-low)/2;

    if(nums[mid] <= target)
        low = mid + 1;
    else
        high = mid;
}

return low - 1;
```
