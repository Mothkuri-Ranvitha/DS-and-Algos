# Peak & Mountain Problems

> Find local maximum using binary search.

---

## Core Idea

Example:

```text
1 3 5 8 7 4 2
      ↑
     Peak
```

Observation:

```text
If slope increasing

Go Right

If slope decreasing

Go Left
```

---

## Recognition Signals

Look for:

* Peak
* Mountain Array
* Local Maximum
* Bitonic Array

---

## Mental Model

```text
Check Slope
      ↓
Move Towards Peak
```

---

## Template

```java
while(low < high){

    int mid =
        low + (high-low)/2;

    if(nums[mid] < nums[mid+1])
        low = mid + 1;

    else
        high = mid;
}
```
