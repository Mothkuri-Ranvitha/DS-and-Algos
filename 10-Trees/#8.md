# Variation 8 – Tree Path & Prefix Sum

> **Pattern Goal:** Solve tree problems where information is carried along the DFS path (sum, path, prefix sum, or number).

---

# Recognition Signals

Whenever the interviewer says:

* Path Sum
* Root to Leaf
* Target Sum
* Return Paths
* Count Paths
* Prefix Sum in Tree
* Root-to-Leaf Number

Think immediately:

> **Carry State During DFS**

---

# Core Mental Model

Unlike previous DFS problems that return information upward, these problems carry information downward.

```text
DFS

↓

Carry State

↓

Update State

↓

Pass to Children
```

The carried state can be:

* Remaining Target
* Current Path
* Running Prefix Sum
* Running Number

---

# Pattern Classification

| Pattern               | State Carried            | Problems |
| --------------------- | ------------------------ | -------- |
| Running Target        | Remaining Sum            | LC112    |
| Running Target + Path | Remaining Sum + Path     | LC113    |
| Prefix Sum            | Running Prefix + HashMap | LC437    |
| Running Number        | Current Decimal Number   | LC129    |

# Problem 1 — LC112 Path Sum

## Recognition Signals

* Root-to-Leaf
* Target Sum
* Does Path Exist

## Core Intuition

Instead of carrying the current sum, carry the **remaining target**.

At each node:

```java
target -= root.val;
```

If a leaf is reached and the remaining target is zero, a valid path exists.

## Mental Model

```text
Target

↓

Subtract Current Node

↓

Leaf?

↓

Remaining == 0
```

## Key Formula

```java
target -= root.val;

if (leaf)
    return target == 0;

return left || right;
```

## Time Complexity

```
O(n)
```

## Space Complexity

```
O(h)
```

## Interview Tips

* Always check the target only at a leaf.
* Use `||`, since one valid path is enough.

## One-Line Revision

> Carry the remaining target during DFS and return true if a leaf is reached with exactly zero remaining.

# Problem 2 — LC113 Path Sum II

## Recognition Signals

* Return All Paths
* Root-to-Leaf Paths
* Collect Valid Paths

## Core Intuition

LC112 + Backtracking.

Carry:

* Remaining Target
* Current Path

Whenever a valid leaf is found, store a copy of the current path.

## Mental Model

```text
Add Node

↓

DFS

↓

Remove Node
```

## Key Formula

```java
path.add(root.val);

target -= root.val;

if (leaf && target == 0)
    result.add(new ArrayList<>(path));

dfs(left);

dfs(right);

path.remove(path.size() - 1);
```

## Time Complexity

```
O(n × h)
```

## Space Complexity

```
O(h)
```

## Interview Tips

* Always store a copy of the path.
* Always backtrack after DFS.

## One-Line Revision

> Carry the current path during DFS, save valid paths, then backtrack before returning.

# Problem 3 — LC437 Path Sum III

## Recognition Signals

* Path Starts Anywhere
* Count Paths
* Downward Paths
* Tree Prefix Sum

## Core Intuition

Treat the current root-to-node path as an array.

Maintain:

* Running Prefix Sum
* HashMap<PrefixSum, Frequency>

At every node:

```text
Need = Current Prefix - Target
```

If the needed prefix exists, we've found one or more valid paths.

## Mental Model

```text
Running Prefix

↓

HashMap

↓

Check Previous Prefix

↓

Backtrack
```

## Key Formula

```java
currSum += root.val;

count += map.getOrDefault(currSum - target, 0);

map.put(currSum,
        map.getOrDefault(currSum, 0) + 1);

dfs(left);

dfs(right);

map.put(currSum,
        map.get(currSum) - 1);
```

## Time Complexity

```
O(n)
```

## Space Complexity

```
O(h)
```

Worst case: `O(n)` for a skewed tree.

## Interview Tips

* Initialize:

```java
map.put(0L, 1);
```

* Backtrack the prefix map after exploring children.
* Use `long` for the running sum.

## One-Line Revision

> Use a prefix-sum HashMap on the current DFS path to count target-sum paths in O(n).

# Problem 4 — LC129 Sum Root to Leaf Numbers

## Recognition Signals

* Digits in Tree
* Root-to-Leaf Numbers
* Decimal Number

## Core Intuition

Carry the current number.

Append the current digit using:

```java
current = current * 10 + root.val;
```

Return the completed number at every leaf.

## Mental Model

```text
Current Number

↓

Append Digit

↓

Leaf

↓

Return Number
```

## Key Formula

```java
current = current * 10 + root.val;

if (leaf)
    return current;

return dfs(left, current)
     + dfs(right, current);
```

## Time Complexity

```
O(n)
```

## Space Complexity

```
O(h)
```

## Interview Tips

* Multiplying by 10 shifts digits left.
* No need to build strings.

## One-Line Revision

> Build the current decimal number during DFS and return the sum of all root-to-leaf numbers.
