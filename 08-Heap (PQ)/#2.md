# 02 - K Closest Problems

---

# Variation Definition

In this variation, we need to find:

```text
K Closest Elements

K Closest Points

K Nearest Values
```

to a target.

---

Suppose:

```java
arr = [1,2,3,4,5]

k = 4

target = 3
```

Closest elements:

```java
[1,2,3,4]
```

---

Observation:

We do NOT need:

```text
All distances
```

We only need:

```text
K Smallest Distances
```

This immediately suggests:

```text
Heap
```

# Core Idea

Convert problem into:

```text
Distance Problem
```

---

Calculate:

```java
distance =
Math.abs(value-target)
```

or

```java
distance =
x*x + y*y
```

---

Then keep:

```text
K Closest Elements
```

inside heap.

---

# Heap Choice

We want:

```text
Smallest Distances
```

Therefore maintain:

```text
Max Heap of Size K
```

Why?

Because the farthest element among current K remains on top.

Whenever a closer element arrives:

```text
Remove Farthest

Insert Closer
```

# Problem 1

# LC973 - K Closest Points to Origin

## Problem Description

Given:

```java
points =
[[1,3],
[-2,2]]

k = 1
```

Distances:

```java
(1,3)

√10
```

```java
(-2,2)

√8
```

Closest:

```java
[-2,2]
```

---

## Intuition

Distance formula:

```java
d² = x² + y²
```

No need for sqrt.

---

Store:

```java
[x,y,distance]
```

inside Max Heap.

---

Heap Top:

```text
Current Farthest Point
```

among K closest.

---

When heap size exceeds K:

```java
pq.poll();
```

Remove farthest.

---

Eventually heap contains:

```text
K Closest Points
```

---





---

Point:

```java

```

```java

```



---

## Algorithm

1. Compute distance.
2. Insert into Max Heap.
3. If size > K:

   * remove farthest.
4. Remaining points are answer.

---

## Code

```java
class Solution {

    public int[][] kClosest(
            int[][] points,
            int k) {

        PriorityQueue<int[]> pq =
            new PriorityQueue<>(
                (a,b) ->
                (b[0]*b[0]+b[1]*b[1])
                -
                (a[0]*a[0]+a[1]*a[1]));

        for(int[] point : points){

            pq.offer(point);

            if(pq.size() > k){

                pq.poll();
            }
        }

        int[][] ans =
            new int[k][2];

        for(int i=0;i<k;i++){

            ans[i] = pq.poll();
        }

        return ans;
    }
}
```

---

## Complexity

```text
Time  : O(N log K)

Space : O(K)
```

# Problem 2

# LC658 - Find K Closest Elements

## Problem Description

Given:

```java
arr = [1,2,3,4,5]

k = 4

x = 3
```

Answer:

```java
[1,2,3,4]
```

---

## Intuition

Distance:

```java
|value - x|
```

---

For:

```java
1
```

Distance:

```java
2
```

---

For:

```java
2
```

Distance:

```java
1
```

---

For:

```java
3
```

Distance:

```java
0
```

---

Keep only:

```text
K Closest Elements
```

inside Max Heap.

---

```java
[value,distance]
```

---


---

```java

```

## Heap Solution Code

```java
class Solution {

    public List<Integer>
        findClosestElements(
            int[] arr,
            int k,
            int x) {

        PriorityQueue<Integer> pq =
            new PriorityQueue<>(
                (a,b)->{

                    int d1 =
                        Math.abs(a-x);

                    int d2 =
                        Math.abs(b-x);

                    if(d1 == d2){

                        return b-a;
                    }

                    return d2-d1;
                });

        for(int num : arr){

            pq.offer(num);

            if(pq.size() > k){

                pq.poll();
            }
        }

        List<Integer> ans =
            new ArrayList<>();

        while(!pq.isEmpty()){

            ans.add(pq.poll());
        }

        Collections.sort(ans);

        return ans;
    }
}
```

# Key Interview Rule

For:

```text
K Largest
```

Use:

```text
Min Heap Size K
```

---

For:

```text
K Closest
K Smallest Distance
```

Use:

```text
Max Heap Size K
```

because we want to throw away:

```text
Farthest Element
```

---

# Decision Formula

```text
Need K Largest
    ↓
Min Heap

Need K Smallest
    ↓
Max Heap

Need K Closest
    ↓
Max Heap
```
