# Greedy Pattern – Variation 3: Reachability & Jump Greedy

## Core Idea

Maintain the **best reachable state** while traversing once.

Instead of exploring every possible path, greedily track the farthest progress.

---

## Recognition Signals

Use this pattern when you see:

- Reach the end
- Minimum jumps
- Can reach destination
- Cover an interval
- Maximum reachable position
- Fuel stations
- Video clips
- Water taps

---

## Mental Model

```text
Current Position
        │
        ▼
How far can I reach?
        │
        ▼
Update Farthest Reach
        │
        ▼
Boundary Finished?
        │
   Yes ─────► Take Next Jump / Resource
```

# Problems

## 1. LC55 – Jump Game ⭐⭐⭐⭐⭐

### Recognition

- Can reach end?
- Maximum jump length

### Intuition

Keep the farthest reachable index.

If `i > farthest`, you are stuck.

### Code

```java
class Solution {
    public boolean canJump(int[] nums) {
        int farthest = 0;

        for (int i = 0; i < nums.length; i++) {
            if (i > farthest)
                return false;

            farthest = Math.max(farthest, i + nums[i]);
        }

        return true;
    }
}
```

**Time:** O(N)

**Space:** O(1)

## 2. LC45 – Jump Game II ⭐⭐⭐⭐⭐

### Recognition

- Minimum jumps
- Reach end with minimum moves

### Intuition

Treat every jump as one BFS level.

Maintain:

- currentEnd
- farthest

When current boundary ends, increase jump count.

### Code

```java
class Solution {

    public int jump(int[] nums) {

        int jumps = 0;
        int currentEnd = 0;
        int farthest = 0;

        for (int i = 0; i < nums.length - 1; i++) {

            farthest = Math.max(farthest, i + nums[i]);

            if (i == currentEnd) {
                jumps++;
                currentEnd = farthest;
            }
        }

        return jumps;
    }
}
```

**Time:** O(N)

**Space:** O(1)

## 3. LC134 – Gas Station ⭐⭐⭐⭐⭐

### Recognition

- Circular route
- Gas & cost
- Find starting station

### Intuition

If total gas < total cost → impossible.

Whenever current tank becomes negative, restart from next station.

### Code

```java
class Solution {

    public int canCompleteCircuit(int[] gas, int[] cost) {

        int total = 0;
        int tank = 0;
        int start = 0;

        for (int i = 0; i < gas.length; i++) {

            int diff = gas[i] - cost[i];

            total += diff;
            tank += diff;

            if (tank < 0) {
                start = i + 1;
                tank = 0;
            }
        }

        return total >= 0 ? start : -1;
    }
}
```

**Time:** O(N)

**Space:** O(1)

## 4. LC1326 – Minimum Number of Taps ⭐⭐⭐⭐

### Recognition

- Cover interval
- Minimum taps

### Intuition

Convert every tap into an interval.

Build a `reach[]` array.

Run the **Jump Game II** greedy algorithm.

### Code

```java
class Solution {

    public int minTaps(int n, int[] ranges) {

        int[] reach = new int[n + 1];

        for (int i = 0; i <= n; i++) {

            int left = Math.max(0, i - ranges[i]);
            int right = Math.min(n, i + ranges[i]);

            reach[left] = Math.max(reach[left], right);
        }

        int taps = 0;
        int currentEnd = 0;
        int farthest = 0;

        for (int i = 0; i < n; i++) {

            farthest = Math.max(farthest, reach[i]);

            if (i == currentEnd) {

                if (farthest == currentEnd)
                    return -1;

                taps++;
                currentEnd = farthest;
            }
        }

        return taps;
    }
}
```

**Time:** O(N)

**Space:** O(N)

## 5. LC1024 – Video Stitching ⭐⭐⭐⭐

### Recognition

- Minimum clips
- Cover interval

### Intuition

Convert clips into a `reach[]` array.

Apply the same greedy algorithm as **Jump Game II**.

### Code

```java
class Solution {

    public int videoStitching(int[][] clips, int time) {

        int[] reach = new int[time + 1];

        for (int[] clip : clips) {

            if (clip[0] <= time) {
                reach[clip[0]] = Math.max(reach[clip[0]],
                        Math.min(time, clip[1]));
            }
        }

        int videos = 0;
        int currentEnd = 0;
        int farthest = 0;

        for (int i = 0; i < time; i++) {

            farthest = Math.max(farthest, reach[i]);

            if (i == currentEnd) {

                if (farthest == currentEnd)
                    return -1;

                videos++;
                currentEnd = farthest;
            }
        }

        return videos;
    }
}
```

**Time:** O(N + time)

**Space:** O(time)
