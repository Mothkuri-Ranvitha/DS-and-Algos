# Greedy Pattern – Variation 5: Scheduling Greedy (Heap Based)

## Core Idea

Scheduling problems usually involve **deadlines**, **events**, **projects**, or **limited resources**.

The common strategy is:

```text
Sort
   ↓
Greedily Select
   ↓
If Constraint Breaks
   ↓
Use Heap to Remove / Choose Best
```

---

# Recognition Signals

Use this pattern when you see:

- Deadlines
- Events
- Projects
- Courses
- Attend maximum
- Finish before deadline
- Maximize profit
- Hire minimum cost

# Problem 1 — LC630 Course Schedule III ⭐⭐⭐⭐⭐

### Recognition

- Courses
- Duration
- Deadline
- Maximum courses

### Intuition

Sort by deadline.

Take every course.

If total time exceeds deadline,

remove the **longest course**.

### Code

```java
class Solution {

    public int scheduleCourse(int[][] courses) {

        Arrays.sort(courses,
                (a,b)->a[1]-b[1]);

        PriorityQueue<Integer> pq =
                new PriorityQueue<>(Collections.reverseOrder());

        int time = 0;

        for(int[] c : courses){

            time += c[0];
            pq.offer(c[0]);

            if(time > c[1])
                time -= pq.poll();
        }

        return pq.size();
    }
}
```

**Time:** O(N log N)

**Space:** O(N)

# Problem 2 — LC1353 Maximum Number of Events ⭐⭐⭐⭐

### Recognition

- Attend events
- One event per day

### Intuition

Sort events by start day.

For every day:

- Add available events.
- Attend event ending earliest.

Use **Min Heap** of ending days.

### Code

```java
class Solution {

    public int maxEvents(int[][] events) {

        Arrays.sort(events,
                (a,b)->a[0]-b[0]);

        PriorityQueue<Integer> pq =
                new PriorityQueue<>();

        int day = 0;
        int i = 0;
        int ans = 0;
        int n = events.length;

        while(i < n || !pq.isEmpty()){

            if(pq.isEmpty())
                day = events[i][0];

            while(i < n &&
                    events[i][0] == day){

                pq.offer(events[i][1]);
                i++;
            }

            while(!pq.isEmpty() &&
                    pq.peek() < day)
                pq.poll();

            if(!pq.isEmpty()){

                pq.poll();
                ans++;
            }

            day++;
        }

        return ans;
    }
}
```

**Time:** O(N log N)

**Space:** O(N)

# Problem 3 — LC502 IPO ⭐⭐⭐⭐⭐

### Recognition

- Projects
- Capital
- Profit
- Choose at most K

### Intuition

Maintain

- Min Heap → Capital
- Max Heap → Profit

Move all affordable projects into max heap.

Always pick highest profit.

### Code

```java
class Solution {

    public int findMaximizedCapital(int k,
                                    int w,
                                    int[] profits,
                                    int[] capital) {

        PriorityQueue<int[]> minCapital =
                new PriorityQueue<>((a,b)->a[0]-b[0]);

        PriorityQueue<Integer> maxProfit =
                new PriorityQueue<>(Collections.reverseOrder());

        for(int i=0;i<profits.length;i++)
            minCapital.offer(new int[]{capital[i],profits[i]});

        while(k-- > 0){

            while(!minCapital.isEmpty()
                    && minCapital.peek()[0] <= w){

                maxProfit.offer(minCapital.poll()[1]);
            }

            if(maxProfit.isEmpty())
                break;

            w += maxProfit.poll();
        }

        return w;
    }
}
```

**Time:** O(N log N)

**Space:** O(N)

# Problem 4 — LC857 Minimum Cost to Hire K Workers ⭐⭐⭐⭐⭐

### Recognition

- Hire workers
- Wage
- Quality
- Minimum cost

### Intuition

Sort by

```text
wage / quality
```

Maintain

- Max Heap of qualities
- Running quality sum

Current ratio determines payment.

### Code

```java
class Solution {

    public double mincostToHireWorkers(int[] quality,
                                       int[] wage,
                                       int k) {

        int n = quality.length;

        double[][] workers =
                new double[n][2];

        for(int i=0;i<n;i++){

            workers[i][0] =
                    (double)wage[i]/quality[i];

            workers[i][1] =
                    quality[i];
        }

        Arrays.sort(workers,
                Comparator.comparingDouble(a->a[0]));

        PriorityQueue<Integer> pq =
                new PriorityQueue<>(Collections.reverseOrder());

        int sumQuality = 0;

        double ans = Double.MAX_VALUE;

        for(double[] w : workers){

            sumQuality += (int)w[1];
            pq.offer((int)w[1]);

            if(pq.size() > k)
                sumQuality -= pq.poll();

            if(pq.size()==k)
                ans = Math.min(ans,
                        sumQuality*w[0]);
        }

        return ans;
    }
}
```

**Time:** O(N log N)

**Space:** O(N)

# One-Line Revision

```text
Sort first.

Choose greedily.

If schedule becomes invalid,
remove the worst task using a heap.

If choosing among available tasks,
pick the best one using a heap.
```
