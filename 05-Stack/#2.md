# Expression Evaluation

> Use a Stack to evaluate mathematical expressions.

---

# Definition

Expression Evaluation problems require computing the result of an arithmetic expression while respecting operator order.

Examples:

```text
2 + 3

2 + 3 * 4

(2 + 3) * 4

2 1 + 3 *
```

---

# Why Stack?

When evaluating an expression, some operations must wait.

Example:

```text
2 + 3 * 4
```

We cannot immediately calculate:

```text
2 + 3
```

because:

```text
*
```

has higher precedence.

We need temporary storage.

Stack provides that storage.

---

# Core Intuition

Think:

```text
Some numbers/operators
must wait until enough
information becomes available.
```

Store them in a stack.

Process later.

# LC150 - Evaluate Reverse Polish Notation

## Problem

Evaluate the value of an arithmetic expression in Reverse Polish Notation.

Valid operators:

```text
+
-
*
/
```

Example:

```text
["2","1","+","3","*"]

((2 + 1) * 3)

Answer = 9
```

---

## Algorithm

1. Create Stack<Integer>.
2. Traverse tokens.
3. If number:

   * Push into stack.
4. If operator:

   * Pop two numbers.
   * Perform operation.
   * Push result back.
5. Final stack top is answer.

---

## Logic

Operator always uses the two most recent numbers.

Example:

```text
2 1 +
```

When '+' appears:

```text
Pop 1
Pop 2

2 + 1 = 3

Push 3
```

---

## Code

```java
class Solution {

    public int evalRPN(String[] tokens) {

        Stack<Integer> st = new Stack<>();

        for(String token : tokens){

            if(token.equals("+")){

                int b = st.pop();
                int a = st.pop();

                st.push(a + b);
            }

            else if(token.equals("-")){

                int b = st.pop();
                int a = st.pop();

                st.push(a - b);
            }

            else if(token.equals("*")){

                int b = st.pop();
                int a = st.pop();

                st.push(a * b);
            }

            else if(token.equals("/")){

                int b = st.pop();
                int a = st.pop();

                st.push(a / b);
            }

            else{

                st.push(Integer.parseInt(token));
            }
        }

        return st.peek();
    }
}
```

Time: O(N)
