# Balanced Parentheses

> The most important Stack pattern.

---

# Definition

Balanced Parentheses problems involve matching opening brackets with their corresponding closing brackets.

Examples:

```text
()

[]

{}

({[]})

((()))
```

Valid pairs must:

1. Open first
2. Close later
3. Close in correct order

---

# Why Does This Pattern Exist?

Consider:

```text
({[]})
```

When we encounter:

```text
(
{
[
```

We cannot verify them immediately.

We must remember them until their matching closing brackets appear.

This is exactly what a Stack does.

```text
Remember Now
     ↓
Use Later
```

Stack<Character> stack =
        new Stack<>();

    if(ch is opening){

        stack.push(ch);
    }

    else{

            return false;

        if(not matching)
            return false;

        stack.pop();
    }

# Java Solution

```java
class Solution {

    public boolean isValid(String s) {

        Stack<Character> stack =
                new Stack<>();

        for(char ch : s.toCharArray()) {

            if(ch=='(' ||
               ch=='{' ||
               ch=='['){ 

                stack.push(ch);
            }

            else{

                if(stack.isEmpty())
                    return false;

                char top =
                        stack.pop();

                if(ch==')' && top!='(')
                    return false;

                if(ch=='}' && top!='{')
                    return false;

                if(ch==']' && top!='[')
                    return false;
            }
        }

        return stack.isEmpty();
    }
}
```

# LC921 - Minimum Add to Make Parentheses Valid

## Problem

Given a string containing only '(' and ')', return the minimum number of parentheses that must be added to make the string valid.

---

## Algorithm

1. Traverse the string.
2. Push '(' into stack.
3. For ')':

   * If matching '(' exists, pop it.
   * Otherwise count one missing '('.
4. After traversal, remaining '(' in stack need matching ')'.
5. Answer = unmatched ')' + unmatched '('.

---

## Logic

Every ')' requires a previous '('.

Every '(' requires a future ')'.

Count all unmatched parentheses.

---

## Code

```java
class Solution {

    public int minAddToMakeValid(String s) {

        Stack<Character> st = new Stack<>();

        int additions = 0;

        for(char ch : s.toCharArray()){

            if(ch == '('){

                st.push(ch);

            }else{

                if(!st.isEmpty()){

                    st.pop();

                }else{

                    additions++;
                }
            }
        }

        return additions + st.size();
    }
}
```

---

## Complexity

Time: O(N)

Space: O(N)

# LC1249 - Minimum Remove to Make Valid Parentheses

## Problem

Given a string containing '(', ')' and lowercase letters, remove the minimum number of invalid parentheses so the resulting string becomes valid.

Return any valid string.

---

## Algorithm

1. Store indices of '(' in stack.
2. For every ')':

   * If matching '(' exists, pop.
   * Otherwise mark ')' for removal.
3. After traversal:

   * Remaining '(' in stack are invalid.
   * Mark them for removal.
4. Build answer excluding removed indices.

---

## Logic

Valid parentheses require matching pairs.

Unmatched ')' are removed immediately.

Unmatched '(' are removed after traversal.

---

## Code

```java
class Solution {

    public String minRemoveToMakeValid(String s) {

        Stack<Integer> stack = new Stack<>();
        HashSet<Integer> remove = new HashSet<>();

        for(int i=0;i<s.length();i++){

            char ch = s.charAt(i);

            if(ch=='('){

                stack.push(i);

            }else if(ch==')'){

                if(!stack.isEmpty()){

                    stack.pop();

                }else{

                    remove.add(i);
                }
            }
        }

        while(!stack.isEmpty()){

            remove.add(stack.pop());
        }

        StringBuilder ans = new StringBuilder();

        for(int i=0;i<s.length();i++){

            if(!remove.contains(i)){

                ans.append(s.charAt(i));
            }
        }

        return ans.toString();
    }
}
```

---

## Complexity

Time: O(N)

Space: O(N)
# LC32 - Longest Valid Parentheses

## Problem

Given a string containing only '(' and ')', return the length of the longest valid parentheses substring.

---

## Algorithm

1. Push -1 into stack.
2. Traverse string.
3. For '(':

   * Push index.
4. For ')':

   * Pop stack.
   * If stack becomes empty:

     * Push current index as boundary.
   * Else:

     * Calculate valid length using:

```java
length = i - stack.peek();
```

5. Maintain maximum length.

---

## Logic

Stack stores:

* Indices of unmatched '('
* Latest invalid boundary

When a valid pair closes:

```java
i - stack.peek()
```

gives current valid substring length.

The boundary prevents invalid ')' from affecting future calculations.

---

## Code

```java
class Solution {

    public int longestValidParentheses(String s) {

        Stack<Integer> st = new Stack<>();

        st.push(-1);

        int maxLen = 0;

        for(int i=0;i<s.length();i++){

            if(s.charAt(i)=='('){

                st.push(i);

            }else{

                st.pop();

                if(st.isEmpty()){

                    st.push(i);

                }else{

                    maxLen =
                        Math.max(
                            maxLen,
                            i - st.peek()
                        );
                }
            }
        }

        return maxLen;
    }
}
```

---

## Complexity

Time: O(N)

Space: O(N)
