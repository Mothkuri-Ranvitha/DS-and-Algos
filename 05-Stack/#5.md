# String Manipulation using Stack

> Pattern: Stack + String Processing

---

# Definition

String Manipulation using Stack is a pattern where we use a stack to process, modify, decode, simplify, or transform strings.

The stack helps when:

```text
Recent characters
must be processed first.
```

---

# Why Does This Pattern Exist?

Many string problems require:

```text
Undo Recent Character

Remove Duplicates

Handle Nested Structures

Decode Patterns

Reverse Portions

Build Final String
```

These operations naturally follow:

```text
Last In
First Out
```

which is exactly how a stack works.

---

# Core Intuition

Whenever a character arrives:

```text
Current Character
       ↓
Needs information from
recent characters
       ↓
Use Stack
```

Think:

```text
Recent characters matter
more than old characters.
```

---

# Recognition Signals

Look for:

```text
Remove Characters

Decode String

Nested Pattern

Reverse Substring

Simplify Path

Adjacent Duplicates

Backspace Processing
```

# Variation 1

# LC1047 Remove All Adjacent Duplicates in String

## Problem

Remove adjacent duplicate characters repeatedly.

Example:

```text
abbaca

bb removed

aaca

aa removed

ca
```

Output:

```text
ca
```

---

## Intuition

If current character equals stack top:

```text
Duplicate Found

Remove Both
```

Otherwise:

```text
Push Current Character
```

---

## Algorithm

```text
Same as top?
→ Pop

Different?
→ Push
```

---

## Code

```java
class Solution {

    public String removeDuplicates(String s) {

        Stack<Character> st =
                new Stack<>();

        for(char ch : s.toCharArray()) {

            if(!st.isEmpty()
                    && st.peek()==ch) {

                st.pop();
            }

            else {

                st.push(ch);
            }
        }

        StringBuilder sb =
                new StringBuilder();

        for(char ch : st) {

            sb.append(ch);
        }

        return sb.toString();
    }
}
```

---

## Complexity

```text
Time  : O(N)

Space : O(N)
```

# Variation 2

# LC1209 Remove All Adjacent Duplicates in String II

## Problem

Remove character groups of size:

```text
k
```

Example:

```text
deeedbbcccbdaa

k = 3
```

Output:

```text
aa
```

---

## Intuition

Need frequency tracking.

Store:

```text
(character,count)
```

in stack.

Whenever:

```text
count == k
```

remove group.

---

## Code

```java
class Solution {

    public String removeDuplicates(
            String s,
            int k) {

        Stack<int[]> st =
                new Stack<>();

        for(char ch :
                s.toCharArray()) {

            if(!st.isEmpty()
                    && st.peek()[0]==ch) {

                st.peek()[1]++;

                if(st.peek()[1]==k) {

                    st.pop();
                }
            }

            else {

                st.push(
                        new int[]{
                                ch,
                                1
                        });
            }
        }

        StringBuilder sb =
                new StringBuilder();

        for(int[] pair : st) {

            for(int i=0;
                i<pair[1];
                i++) {

                sb.append(
                        (char)pair[0]
                );
            }
        }

        return sb.toString();
    }
}
```

---

## Complexity

```text
Time  : O(N)

Space : O(N)
```

# Variation 3

# LC394 Decode String

## Problem

Example:

```text
3[a2[c]]
```

Output:

```text
accaccacc
```

---

## Intuition

Whenever:

```text
]
```

appears,

decode the most recent block.

Stack naturally handles nesting.

---

## Mental Model

```text
3[a2[c]]

Inner

2[c]
↓

cc

Outer

3[acc]
↓

accaccacc
```

---

## Code

```java
class Solution {

    public String decodeString(
            String s) {

        Stack<Integer> countStack =
                new Stack<>();

        Stack<StringBuilder> strStack =
                new Stack<>();

        StringBuilder current =
                new StringBuilder();

        int num = 0;

        for(char ch :
                s.toCharArray()) {

            if(Character.isDigit(ch)) {

                num =
                    num*10+(ch-'0');
            }

            else if(ch=='[') {

                countStack.push(num);

                strStack.push(current);

                current =
                    new StringBuilder();

                num = 0;
            }

            else if(ch==']') {

                int repeat =
                        countStack.pop();

                StringBuilder prev =
                        strStack.pop();

                while(repeat-- > 0) {

                    prev.append(current);
                }

                current = prev;
            }

            else {

                current.append(ch);
            }
        }

        return current.toString();
    }
}
```

---

## Complexity

```text
Time  : O(N)

Space : O(N)
```

# Variation 4

# LC71 Simplify Path

## Problem

Convert:

```text
/home//foo/
```

to:

```text
/home/foo
```

Handle:

```text
.

..
```

---

## Intuition

Directory navigation behaves like stack.

```text
folder
→ Push

..
→ Pop
```

---

## Code

```java
class Solution {

    public String simplifyPath(
            String path) {

        Stack<String> st =
                new Stack<>();

        String[] parts =
                path.split("/");

        for(String p : parts) {

            if(p.equals("")
                    || p.equals(".")) {

                continue;
            }

            if(p.equals("..")) {

                if(!st.isEmpty()) {

                    st.pop();
                }
            }

            else {

                st.push(p);
            }
        }

        StringBuilder sb =
                new StringBuilder();

        for(String dir : st) {

            sb.append("/")
              .append(dir);
        }

        return sb.length()==0
                ? "/"
                : sb.toString();
    }
}
```

---

## Complexity

```text
Time  : O(N)

Space : O(N)
```

# LC1190 Reverse Substrings Between Parentheses

## Problem

Example:

```text
(a(bc)d)
```


---

## Intuition

When:

```text
)
```

Stack helps find the nearest opening bracket.

---

## Algorithm

```text
Push Characters

Pop until '('

Reverse

Push back
```


# Variation 6

# LC402 Remove K Digits

## Problem

Example:

```text
1432219

k = 3
```

Output:

```text
1219
```

---

## Intuition

To get smallest number:

remove larger digits appearing before smaller digits.

Example:

```text
4 before 3

Remove 4
```

Stack maintains increasing digits.

---

## Mental Model

```text
1432219

4 > 3

Remove 4

132219

3 > 2

Remove 3

12219

2 > 1

Remove 2

1219
```

---

## Code

```java
class Solution {

    public String removeKdigits(
            String num,
            int k) {

        Stack<Character> st =
                new Stack<>();

        for(char ch :
                num.toCharArray()) {

            while(!st.isEmpty()
                    && k>0
                    && st.peek()>ch) {

                st.pop();
                k--;
            }

            st.push(ch);
        }

        while(k>0) {

            st.pop();
            k--;
        }

        StringBuilder sb =
                new StringBuilder();

        for(char ch : st) {

            sb.append(ch);
        }

        while(sb.length()>0
                && sb.charAt(0)=='0') {

            sb.deleteCharAt(0);
        }

        return sb.length()==0
                ? "0"
                : sb.toString();
    }
}
```

---

## Complexity

```text
Time  : O(N)

Space : O(N)
```
