# Variation 5 — K-Sum

---

## Intuition

Every K-Sum reduces to 2Sum at the bottom.
Fix outer elements with loops. Run two pointers on what remains.

```
3Sum → fix 1 element  + 2Sum inside → O(N²)
4Sum → fix 2 elements + 2Sum inside → O(N³)
```

Always sort first. Sorting enables two pointers, duplicate skipping, and early termination.

---

## LC15 — 3Sum
Find all unique triplets summing to zero.

```java
List<List<Integer>> threeSum(int[] nums) {
    Arrays.sort(nums);
    List<List<Integer>> res = new ArrayList<>();

    for (int i = 0; i < nums.length - 2; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue; // skip duplicate
        if (nums[i] > 0) break;                       // early exit

        int L = i+1, R = nums.length-1, target = -nums[i];
        while (L < R) {
            int sum = nums[L] + nums[R];
            if (sum == target) {
                res.add(Arrays.asList(nums[i], nums[L], nums[R]));
                while (L < R && nums[L] == nums[L+1]) L++;
                while (L < R && nums[R] == nums[R-1]) R--;
                L++; R--;
            }
            else if (sum < target) L++;
            else                   R--;
        }
    }
    return res;
}
```

## LC16 — 3Sum Closest
No duplicate handling needed. Track minimum absolute difference.

```java
int threeSumClosest(int[] nums, int target) {
    Arrays.sort(nums);
    int closest = nums[0] + nums[1] + nums[2];

    for (int i = 0; i < nums.length - 2; i++) {
        int L = i+1, R = nums.length-1;
        while (L < R) {
            int sum = nums[i] + nums[L] + nums[R];
            if (Math.abs(sum - target) < Math.abs(closest - target))
                closest = sum;
            if      (sum < target) L++;
            else if (sum > target) R--;
            else    return sum;
        }
    }
    return closest;
}
```

## LC18 — 4Sum
Two nested loops, then 2Sum. Cast to `long` to avoid overflow.

```java
List<List<Integer>> fourSum(int[] nums, int target) {
    Arrays.sort(nums);
    List<List<Integer>> res = new ArrayList<>();
    int n = nums.length;

    for (int i = 0; i < n-3; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue;
        for (int j = i+1; j < n-2; j++) {
            if (j > i+1 && nums[j] == nums[j-1]) continue;
            int L = j+1, R = n-1;
            long need = (long) target - nums[i] - nums[j];
            while (L < R) {
                long sum = nums[L] + nums[R];
                if (sum == need) {
                    res.add(Arrays.asList(nums[i],nums[j],nums[L],nums[R]));
                    while (L < R && nums[L] == nums[L+1]) L++;
                    while (L < R && nums[R] == nums[R-1]) R--;
                    L++; R--;
                }
                else if (sum < need) L++;
                else                 R--;
            }
        }
    }
    return res;
}
```

## LC259 — 3Sum Smaller
Count triplets with sum < target. When `sum < target`, all pairs between L and R are valid.

```java
int threeSumSmaller(int[] nums, int target) {
    Arrays.sort(nums);
    int count = 0;
    for (int i = 0; i < nums.length-2; i++) {
        int L = i+1, R = nums.length-1;
        while (L < R) {
            if (nums[i] + nums[L] + nums[R] < target) {
                count += R - L;  // all pairs (L..R-1, R) valid
                L++;
            } else R--;
        }
    }
    return count;
}
```

## LC611 — Valid Triangle Number
Fix largest side at R. Count pairs (L, M) where `a + b > c`.

```java
int triangleNumber(int[] nums) {
    Arrays.sort(nums);
    int count = 0;
    for (int R = nums.length-1; R >= 2; R--) {
        int L = 0, M = R-1;
        while (L < M) {
            if (nums[L] + nums[M] > nums[R]) {
                count += M - L;
                M--;
            } else L++;
        }
    }
    return count;
}
```

## Duplicate Skipping Rule

```java
// Outer element — check backward
if (i > 0 && nums[i] == nums[i-1]) continue;

// Inner pointers after finding answer
while (L < R && nums[L] == nums[L+1]) L++;
while (L < R && nums[R] == nums[R-1]) R--;
L++; R--;
```
