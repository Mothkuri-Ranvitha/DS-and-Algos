# Variation 1 — Palindrome Check

---

## Intuition

Compare characters from both ends. Move inward. Stop when they don't match.

```
"r a c e c a r"
  L         R   → r == r ✓
    L     R     → a == a ✓
      L R       → c == c ✓
        L=R     → done → palindrome
```

---

## Template

```java
int L = 0, R = s.length() - 1;
while (L < R) {
    if (s.charAt(L) != s.charAt(R)) return false;
    L++; R--;
}
return true;
```

## Sub-variations

| Type | Extra step | Problem |
|------|-----------|---------|
| Basic compare | None | LC9 |
| Skip non-alphanumeric | Skip junk before comparing | LC125 |
| One deletion allowed | On mismatch, try skipping L or R | LC680 |
| Find longest | Expand from center instead | LC5 |

---

## LC125 — Valid Palindrome
Skip spaces and symbols. Compare only letters and digits.

```java
boolean isPalindrome(String s) {
    int L = 0, R = s.length() - 1;
    while (L < R) {
        while (L < R && !Character.isLetterOrDigit(s.charAt(L))) L++;
        while (L < R && !Character.isLetterOrDigit(s.charAt(R))) R--;
        if (Character.toLowerCase(s.charAt(L)) !=
            Character.toLowerCase(s.charAt(R))) return false;
        L++; R--;
    }
    return true;
}
```

---

## LC680 — Valid Palindrome II
Allow at most one deletion. On mismatch, try skipping either side.

```java
boolean validPalindrome(String s) {
    int L = 0, R = s.length() - 1;
    while (L < R) {
        if (s.charAt(L) != s.charAt(R))
            return isPalin(s, L+1, R) || isPalin(s, L, R-1);
        L++; R--;
    }
    return true;
}

boolean isPalin(String s, int L, int R) {
    while (L < R) {
        if (s.charAt(L) != s.charAt(R)) return false;
        L++; R--;
    }
    return true;
}
```

---

## LC5 — Longest Palindromic Substring
Don't start from ends — expand outward from every center.
Two cases: odd length `(i, i)` and even length `(i, i+1)`.

```java
String longestPalindrome(String s) {
    int start = 0, maxLen = 1;
    for (int i = 0; i < s.length(); i++) {
        int len = Math.max(expand(s, i, i), expand(s, i, i+1));
        if (len > maxLen) {
            maxLen = len;
            start = i - (len - 1) / 2;
        }
    }
    return s.substring(start, start + maxLen);
}

int expand(String s, int L, int R) {
    while (L >= 0 && R < s.length() && s.charAt(L) == s.charAt(R)) {
        L--; R++;
    }
    return R - L - 1;
}
```
