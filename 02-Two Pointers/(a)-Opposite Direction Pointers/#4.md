# Variation 4 — Maximize / Minimize Area

---

## Intuition

Two boundaries. Something fills between them.
Water level is always determined by the **shorter** boundary.

```
Area = min(height[L], height[R]) × (R - L)
```

Always move the shorter side. Moving the taller side can never improve the answer — width shrinks and height stays the same or gets worse.

---

## Template

```java
int L = 0, R = height.length - 1;
int max = 0;

while (L < R) {
    int area = Math.min(height[L], height[R]) * (R - L);
    max = Math.max(max, area);
    if (height[L] < height[R]) L++;
    else                        R--;
}
return max;
```

---

## LC11 — Container With Most Water
Find two lines that hold the most water.

```
[1, 8, 6, 2, 5, 4, 8, 3, 7]
 L                         R

L=0 R=8 → min(1,7)×8 = 8   → move L (shorter)
L=1 R=8 → min(8,7)×7 = 49  → move R (shorter)
L=1 R=7 → min(8,3)×6 = 18  → move R
...
Answer: 49
```

```java
int maxArea(int[] height) {
    int L = 0, R = height.length - 1, max = 0;
    while (L < R) {
        max = Math.max(max, Math.min(height[L], height[R]) * (R - L));
        if (height[L] < height[R]) L++;
        else                        R--;
    }
    return max;
}
```

## LC42 — Trapping Rain Water
Water at each position = `min(maxLeft, maxRight) - height[i]`.
Track running max on both sides. Always process the side with the smaller running max.

```java
int trap(int[] height) {
    int L = 0, R = height.length - 1;
    int maxL = 0, maxR = 0, water = 0;

    while (L < R) {
        if (height[L] < height[R]) {
            if (height[L] >= maxL) maxL = height[L];
            else water += maxL - height[L];
            L++;
        } else {
            if (height[R] >= maxR) maxR = height[R];
            else water += maxR - height[R];
            R--;
        }
    }
    return water;
}
```

---

## LC11 vs LC42

| | LC11 | LC42 |
|--|------|------|
| Count | Two chosen lines | Every bar contributes |
| Formula | `min(h[L], h[R]) × width` | `min(maxL, maxR) - h[i]` |
| Track | Current heights | Running max both sides |
