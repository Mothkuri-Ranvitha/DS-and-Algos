# Variation 2 — Reverse

---

## Intuition

Swap elements from both ends. Move inward after each swap.

```
[1, 2, 3, 4, 5]
 L           R  → swap → [5, 2, 3, 4, 1]
    L     R     → swap → [5, 4, 3, 2, 1]
       L=R      → done
```

---

## Template

```java
int L = 0, R = arr.length - 1;
while (L < R) {
    swap(arr, L, R);
    L++; R--;
}
```

Only thing that changes across problems: **what counts as swappable**.

---

## LC344 — Reverse String
Pure swap. Nothing to skip.

```java
void reverseString(char[] s) {
    int L = 0, R = s.length - 1;
    while (L < R) {
        char temp = s[L];
        s[L] = s[R];
        s[R] = temp;
        L++; R--;
    }
}
```

---

## LC345 — Reverse Vowels of a String
Skip consonants. Swap only when both pointers land on vowels.

```java
String reverseVowels(String s) {
    char[] arr = s.toCharArray();
    String vowels = "aeiouAEIOU";
    int L = 0, R = arr.length - 1;

    while (L < R) {
        while (L < R && vowels.indexOf(arr[L]) == -1) L++;
        while (L < R && vowels.indexOf(arr[R]) == -1) R--;
        if (L < R) {
            char temp = arr[L];
            arr[L] = arr[R];
            arr[R] = temp;
            L++; R--;
        }
    }
    return new String(arr);
}
```

## LC151 — Reverse Words in a String
Can't swap characters. Split into words, then reverse the word array.

```
"  the sky is blue  "
→ split  → ["the", "sky", "is", "blue"]
→ reverse → ["blue", "is", "sky", "the"]
→ join   → "blue is sky the"
```

```java
String reverseWords(String s) {
    String[] words = s.trim().split("\\s+");
    int L = 0, R = words.length - 1;
    while (L < R) {
        String temp = words[L];
        words[L] = words[R];
        words[R] = temp;
        L++; R--;
    }
    return String.join(" ", words);
}
```

---

## Key Difference

| Problem | Swappable unit | Skip condition |
|---------|---------------|----------------|
| LC344 | Every character | None |
| LC345 | Vowels only | Skip consonants |
| LC151 | Whole words | None |

---

## Complexity

| Problem | Time | Space |
|---------|------|-------|
| LC344 | O(N) | O(1) |
| LC345 | O(N) | O(N) — char array |
| LC151 | O(N) | O(N) — word array |
