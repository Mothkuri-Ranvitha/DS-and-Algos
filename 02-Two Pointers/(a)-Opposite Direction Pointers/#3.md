# Variation 3 — Pair Sum on Sorted Array

---

## Intuition

Array is sorted. Start from both ends.

```
sum too small → move L right  (need bigger value)
sum too big   → move R left   (need smaller value)
sum == target → found
```

Sorted order tells you exactly which direction to move. No guessing.

---

## Template

```java
int L = 0, R = nums.length - 1;
while (L < R) {
    int sum = nums[L] + nums[R];
    if      (sum == target) return result;
    else if (sum < target)  L++;
    else                    R--;
}
```

## LC167 — Two Sum II
Array already sorted. Find pair that sums to target. Return 1-indexed positions.

```
[2, 7, 11, 15]   target = 9

L=0 R=3 → 2+15=17 > 9 → R--
L=0 R=2 → 2+11=13 > 9 → R--
L=0 R=1 → 2+7=9 == 9  → return [1, 2] ✓
```

```java
int[] twoSum(int[] numbers, int target) {
    int L = 0, R = numbers.length - 1;
    while (L < R) {
        int sum = numbers[L] + numbers[R];
        if      (sum == target) return new int[]{L+1, R+1};
        else if (sum < target)  L++;
        else                    R--;
    }
    return new int[]{};
}
```

## LC977 — Squares of Sorted Array
After squaring, largest values are at both edges (most negative or most positive).
Fill result array from right to left — largest square goes in last.

```
[-4, -1, 0, 3, 10]
  L               R

16 vs 100 → 100 is bigger → put at end, R--
16 vs 9   → 16 is bigger  → put at end, L++
...
```

```java
int[] sortedSquares(int[] nums) {
    int n = nums.length;
    int[] result = new int[n];
    int L = 0, R = n - 1, pos = n - 1;

    while (L <= R) {
        int left  = nums[L] * nums[L];
        int right = nums[R] * nums[R];
        if (left > right) { result[pos--] = left;  L++; }
        else              { result[pos--] = right; R--; }
    }
    return result;
}
```

Note: `L <= R` here — need to process the middle element too.

---

## Key Difference

| | LC167 | LC977 |
|--|-------|-------|
| Goal | Find a pair | Transform all elements |
| Fill direction | N/A | Right to left |
| Loop condition | `L < R` | `L <= R` |
