# Move Elements

> Gather useful elements first, move unwanted elements later.

---

## Core Idea

Instead of removing values:

```text
Move them to one side.
```

---

## Mental Model

```text
Fast Finds

↓

Slow Collects

↓

Remaining positions
handled later
```

---

## Recognition Signals

* Move zeroes
* Even/Odd separation
* Group elements
* In-place rearrangement

---

## Example Problems

### LC283 - Move Zeroes

Example:

```text
[0,1,0,3,12]

↓

[1,3,12,0,0]
```

```java
class Solution {
    public void moveZeroes(int[] nums) {

        int slow = 0;

        for(int fast = 0; fast < nums.length; fast++) {

            if(nums[fast] != 0) {

                int temp = nums[slow];
                nums[slow] = nums[fast];
                nums[fast] = temp;

                slow++;
            }
        }
    }
}
```

Time: O(N)

Space: O(1)

### LC905 - Sort Array By Parity

Example:

```text
[3,1,2,4]

↓

[2,4,3,1]
```

```java
class Solution {
    public int[] sortArrayByParity(int[] nums) {

        int slow = 0;

        for(int fast = 0; fast < nums.length; fast++) {

            if(nums[fast] % 2 == 0) {

                int temp = nums[slow];
                nums[slow] = nums[fast];
                nums[fast] = temp;

                slow++;
            }
        }

        return nums;
    }
}
```

Time: O(N)

Space: O(1)
