## Example Problems

### LC75 - Sort Colors

Example:

```text
[2,0,2,1,1,0]

↓

[0,0,1,1,2,2]
```

```java
class Solution {

    public void sortColors(int[] nums) {

        int low = 0;
        int mid = 0;
        int high = nums.length - 1;

        while(mid <= high) {

            if(nums[mid] == 0) {

                swap(nums, low, mid);
                low++;
                mid++;

            } else if(nums[mid] == 1) {

                mid++;

            } else {

                swap(nums, mid, high);
                high--;
            }
        }
    }

    private void swap(int[] nums, int i, int j) {

        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
}
```

Time: O(N)

Space: O(1)
